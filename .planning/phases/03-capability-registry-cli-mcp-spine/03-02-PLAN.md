    ```python
        cap = get_registry().get("workspace.validate")

        # result is a ValidationReport with .errors and .warnings
        for finding in result.errors:

        for finding in result.warnings:
            typer.echo(f"WARNING {finding.path}: {finding.message}")
        typer.echo(f"Validation complete: {len(result.errors)} error(s), {len(result.warnings)} warning(s)")

    ```

    ```python
    @app.command()
    def status(path: Path) -> None:

        items = cap.handler(path)  # returns list of WorkspaceItem
        categories = {"canonical": "Canonical", "support": "Support", "derived": "Derived"}
        for item in items:
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args (user → cli.py) | User-supplied command line arguments parsed by Typer |
| Registry dispatch (cli.py → capabilities) | Command execution passes through registry.get() to callable handlers |
| PIPE handlers (graph_status → workspace files) | Filesystem read operations on workspace directories |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-05 | T (Tampering) | cli.py Typer argument parsing | mitigate | Typer enforces type coercion (int bounds with min/max, string types). Per D-14, validation returns structured errors before any write. |
| T-03-06 | E (Elevation) | registry.get() KeyError handling | mitigate | All registry lookups wrapped in try/except KeyError, display error message and exit with code 1. No unhandled KeyError reaches user. |
| T-03-07 | I (Info Disclosure) | graph_status returns file/domain names | accept | Domain names and card counts from the local workspace filesystem. No sensitive data. Local-only architecture — same user has filesystem access. |
| T-03-08 | T (Tampering) | cli.py still imports init directly | accept | `init` command has interactive prompts (typer.prompt) that don't fit the registry pattern. It's called directly as before — no regression. |
</threat_model>
