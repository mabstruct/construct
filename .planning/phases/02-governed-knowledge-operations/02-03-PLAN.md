    Function `route_source_to_domain(workspace_root: str | Path, source_path: str | Path, domain_hint: str | None = None) -> OperationResult`:
    
    Purpose: Route a source file from workspace inbox/ to its target domain directory, creating a ref entry and organizing the raw file.

    Steps:
    1. Resolve source_path relative to workspace_root
    2. Verify source file exists
    3. Load domains.yaml via WorkspaceLoader(workspace_root).load_domains_registry()
    4. If domain_hint provided AND domain exists in registry → use it
    5. If no domain_hint: Find best matching domain by scanning domain descriptions/content_categories (simple substring/prefix matching on filename)
    6. If no match found → return OperationResult with suggestion to create a domain (per D-11)
    7. Create target directory: `{domain}/inbox/raw/`
    8. Generate ref ID from filename (kebab-case, dedup)
    9. Copy source file to `{domain}/inbox/raw/{original_filename}`
    10. Create ReferenceRecord:
        ```python
        ref = ReferenceRecord(
            id=ref_id,
            title=Path(source_path).stem.replace("-", " ").title(),
            url=f"inbox://{Path(source_path).name}",
            relevance_score=0.5,
            source_tier=5,
            extraction_status=ExtractionStatus.partial,
            ingested_date=today,
            domain=matched_domain,
            search_cluster="manual-ingest",
        )
        ```
    11. Validate via `validate_ref_write(ref.model_dump(), relative_path=f"refs/{ref_id}.json")`
    12. Write ref JSON to `refs/{ref_id}.json`
    13. Log event: action="ingest_paper", target=ref_id
    14. Return OperationResult with ref_id, domain, file paths

    Helper function `_suggest_domain(registry: DomainsRegistry, filename: str) -> str | None`:
    - Simple heuristic: scan domain names, descriptions, and content_categories for substring matches against the filename
    - Return first match, or None

    Helper function `_generate_ref_id(title: str, workspace_root: Path) -> str`:
    - Kebab-case from title
    - Dedup against existing refs/*.json files
    - If duplicate, append -2, -3, etc.

    def test_route_source_to_domain(workspace_path):
        """Source file in inbox gets routed to matching domain."""
        from construct.services.init import DomainInitInput, initialize_workspace
        domain = DomainInitInput(
            domain_id="physics", display_name="Physics", scope="Test",
            taxonomy_seeds=["quantum"], source_priorities=["web"],
            research_seeds=["quantum"],
        )
        initialize_workspace(workspace_path, domain)
        
        # Create inbox source file
        inbox_dir = workspace_path / "inbox"
        inbox_dir.mkdir(exist_ok=True)
        source_file = inbox_dir / "quantum-entanglement-paper.pdf"
        source_file.write_text("dummy content")
        
        result = route_source_to_domain(workspace_path, source_file)
        assert result.success
        assert result.data["domain"] == "physics"
        # Verify file was routed
        assert (workspace_path / "physics" / "inbox" / "raw" / "quantum-entanglement-paper.pdf").exists()
