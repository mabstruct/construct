    **Card Create** — `create_card(workspace_root: str | Path, card_data: dict[str, Any], author: CardAuthor = CardAuthor.construct) -> OperationResult`:
    1. Set `created` to today if not provided, `author` if not provided
    2. Validate card_data via `validate_card_write()` — but need to convert dict to markdown first. Use:
       - Convert dict to YAML frontmatter via ruamel.yaml
       - Build markdown string: `"---\n{frontmatter}\n---\n\n## Summary\n\n{summary}\n\n## Evidence\n\n## Significance\n\n## Open Questions\n"`
       - Call `validate_card_write(markdown)`
    3. If validation fails: catch `ArtifactValidationError` / `ValidationError`, return `OperationResult(success=False, errors=[OperationError(field=..., reason=str(exc), suggestion="Check the card fields and try again.")])`
    4. Generate card ID if not provided (kebab-case from title, dedup against existing)
    5. Ensure `cards/` directory exists
    6. Write card markdown to `cards/{id}.md`
    7. Log event: `append_card_event(workspace_root, EventAgent.construct, "create_card", card_id, f"created {epistemic_type} card", EventResult.success)`
    8. Return `OperationResult(success=True, message=f"Card '{title}' created as {id}", data={"id": id, "path": f"cards/{id}.md"})`

    Use ruamel.yaml to write the YAML frontmatter (preserve field order, no aliases):
    ```python
    _yaml = YAML()
    _yaml.default_flow_style = False
    _yaml.indent(mapping=2, sequence=4)
    ```

    **Card Edit** — `edit_card(workspace_root: str | Path, card_id: str, updates: dict[str, Any], author: CardAuthor = CardAuthor.curator) -> OperationResult`:
    1. Read existing card file at `cards/{card_id}.md`
    2. Parse with `parse_card_markdown()` to get existing card + body
    3. Apply updates to card dict (field-level, do NOT wipe unmentioned fields)
    4. Validate updated card dict (convert to markdown, validate via `validate_card_write`)
    5. If validation fails: return OperationResult with errors
    6. Write updated markdown to file
    7. Log event: `append_card_event(workspace_root, ..., "edit_card", card_id, f"updated fields: {list(updates.keys())}", ...)`
    8. Return OperationResult with success

    **Card Archive** — `archive_card(workspace_root: str | Path, card_id: str, author: CardAuthor = CardAuthor.curator) -> OperationResult`:
    1. Read and parse existing card
    2. Set `lifecycle` to `Lifecycle.archived`
    3. Keep `connects_to` intact (per D-06 — archive preserves connections)
    4. Validate and write
    5. Log event: `append_card_event(workspace_root, ..., "archive_card", card_id, "lifecycle set to archived", ...)`
    6. Return OperationResult

    **Connection Add** — `add_connection(workspace_root: str | Path, from_id: str, to_id: str, conn_type: ConnectionType, note: str | None = None, created_by: ConnectionAuthor = ConnectionAuthor.construct) -> OperationResult`:
    1. Load existing connections.json via `WorkspaceLoader(workspace_root).load_connections()`
    2. Validate `from_id` and `to_id` are kebab-case card IDs (regex check)
    3. Check both card files exist at `cards/{from_id}.md` and `cards/{to_id}.md` (orphan edge prevention)
    4. Check for duplicate (same from+to+type) — if exists, return OperationResult with message "Connection already exists"
    5. Validate `conn_type` is valid `ConnectionType` enum
    6. Create `ConnectionRecord(from_=from_id, to=to_id, type=conn_type, note=note, created=today, created_by=created_by)`
    7. Create updated `ConnectionsFile` with new connection added
    8. Validate via `validate_connections_write()` (catches duplicates, invalid types)
    9. Write updated connections.json
    10. Log event with action="add_connection"
    11. Return OperationResult

    **Connection Remove** — `remove_connection(workspace_root: str | Path, from_id: str, to_id: str, conn_type: ConnectionType) -> OperationResult`:
    1. Load connections.json
    2. Find matching connection (from+to+type)
    3. If not found: return OperationResult(success=False, message="Connection not found")
    4. Remove the connection from the list
    5. Validate and write connections.json
    6. Log event with action="remove_connection"
    7. Return OperationResult

    **Connection List** — `list_connections(workspace_root: str | Path, card_id: str | None = None, include_archived: bool = False) -> OperationResult`:
    1. Load connections.json
    2. If card_id is provided, filter connections where from==card_id OR to==card_id
    3. If include_archived is True, return all connections including those involving archived cards
    4. If include_archived is False, check each connection's card IDs against archive status. Filter out connections where either endpoint is archived.
    5. Return OperationResult with data=list of ConnectionRecord dicts
