<objective>
Build a local Streamlit ops dashboard with three panels: Dashboard (graph health), Gate Review (Q&A + bridge candidates), and Capability Runner (execute capabilities from UI). All capability executions go through the capability registry per D-04.

**Purpose:** ADV-04 requires that "maintainer can exercise capabilities through a local ops UI spike without making that spike the source of truth." The Streamlit app provides a browser-based ops console for running capabilities, reviewing L2 gate results, and monitoring workspace health — without writing to SOT directly.

**Output:** Streamlit app at `src/construct/ui/streamlit_app.py` with 5 pages (Home, Capability Runner, Gate Review, Workflow, Event Tail), sidebar config for workspace/LLM settings, and capability registry integration.
</objective>
