Four Stages

1

Content Hunt

~1 minute
  • Read search-seeds.json
  • Run 10 searches (8 cluster + 2 wildcard)
  • Collect up to 10 candidates per search
  • Write digest markdown
  • Update seen-urls.json
2

Ingestion

~30 seconds
  • Run digest_ingest.py script
  • Create JSON in refs/digest/
  • Resolve via Semantic Scholar
  • Update _progress.json
3

Artifact Enrichment

~1.5 minutes
  • Check Papers With Code
  • Query HuggingFace Hub
  • Fetch GitHub metadata
  • Compose descriptions
  • Write artifacts array to JSON
4

Digest Output

~10 seconds
  • Format Slack-friendly digest
  • Append notable artifacts
  • Final validation

Rate Limits

Service Rate Limit Strategy
Semantic Scholar 1 request per 3 seconds Batch requests; queue resolution calls
Papers With Code 1 request per 2 seconds Prioritize high-confidence papers only
HuggingFace Hub 1 request per 1 second Cache model cards; batch queries
GitHub API 60 requests/hour; 1 per 2 seconds Use authentication for higher limits; prioritize starred repos
